# Flood Prediction System

## Project Overview
This project predicts flood risk using Machine Learning techniques.

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn

## Machine Learning Models
- Decision Tree
- Random Forest
- K-Nearest Neighbors
- XGBoost

## Best Model
XGBoost Accuracy: 93.24%

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

python app.py

3. Open browser

http://127.0.0.1:5000
